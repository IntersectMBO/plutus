    1 -- | This module exports everything from the stdlib via a data type which allows to test
    2 -- various procedures (pretty-printing, type checking, etc) over the entire stdlib in a
    3 -- convenient way: each time a function / data type is added to the stdlib, none of the
    4 -- tests is required to be adapted, instead you just add the new definition to 'stdLib'
    5 -- defined below and all the tests see it automatically.
    6 
    7 {-# LANGUAGE ScopedTypeVariables #-}
    8
