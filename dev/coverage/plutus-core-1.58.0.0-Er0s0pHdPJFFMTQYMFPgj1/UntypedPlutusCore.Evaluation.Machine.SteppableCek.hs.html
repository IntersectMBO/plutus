   67 {-| Evaluate a term using the Steppable CEK machine with logging enabled and keep track of costing.
   68 A wrapper around the internal runCek to debruijn input and undebruijn output.
   69 *THIS FUNCTION IS PARTIAL if the input term contains free variables* -}
   70 runCek
   71   :: ThrowableBuiltins uni fun
   72   => MachineParameters CekMachineCosts fun (CekValue uni fun ann)
   73   -> ExBudgetMode cost uni fun
   74   -> EmitterMode uni fun
   75   -> Term Name uni fun ann
   76   -> CekReport cost Name uni fun
   77 runCek = Common.runCek S.runCekDeBruijn
