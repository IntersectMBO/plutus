## Term Constants

```
data TermCon {Φ} : Φ ⊢⋆ * → Set where
  tmInteger    :
      (i : ℤ)
    → TermCon (con integer)
  tmBytestring :
      (b : U.ByteString)
    → TermCon (con bytestring)
  tmString     :
      (s : String)
    → TermCon (con string)
  tmBool       :
      (b : Bool)
    → TermCon (con bool)
  tmUnit       : TermCon (con unit)
  tmData      : U.DATA → TermCon (con pdata)
```
