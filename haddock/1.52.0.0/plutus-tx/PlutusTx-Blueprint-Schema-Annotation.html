An annotation for the "title" schema attribute.

This annotation could be attached to a type or constructor: {-# ANN type MyFoo (SchemaTitle "My Foo Title") #-} {-# ANN MkMyFoo (SchemaTitle Title) #-} newtype MyFoo = MkMyFoo Int
