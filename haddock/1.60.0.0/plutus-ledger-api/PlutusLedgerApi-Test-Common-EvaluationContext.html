Synopsis
  • type MCostModel = BuiltinCostModel -> MCostModel #

    A helper function to lift to a "full" MCostModel, by mapping *all* of its fields to Just. The fields can be later on cleared, by assigning them to extractCostModelParams to make a mapping of params not in alphabetical order, but in the ParamName order, i.e. the order expected by the ledger.

    Here, overconstrained to MCostModel, but it could also work with `CostModel mcosts bcosts`.
