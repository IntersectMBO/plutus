### Fixed

- Added second constructor to `BuiltinData` to prevent GHC's simplifier from
  exposing `PlutusCore.Data.Data` in join point types, which caused the plugin
  to crash with `Unsupported feature: Type constructor: GHC.Prim.Addr#`.
  Costs up to ~1.3% CPU / ~2.6% memory on GHC 9.12-compiled scripts using
  data-backed containers, because join points over `BuiltinData` are no longer
  eliminated.
